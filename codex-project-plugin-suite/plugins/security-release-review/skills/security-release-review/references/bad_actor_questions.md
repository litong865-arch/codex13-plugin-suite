# 坏人视角问题库

上线前让智能体逐项回答，并在代码中找证据。

## 成本攻击

- 我能不能用脚本无限触发短信、邮件、AI 调用、外部 API 或支付创建？
- 有没有用户级、IP 级、目标级、全局预算和供应商账单告警？
- 有没有低成本方式绕过登录、验证码或设备限制？

## UGC 与内容占位

- 我能不能把昵称、头像、简介、评论、留言、帖子、私信用作广告位？
- 新用户能不能提交即公开？
- 能不能塞链接、二维码、HTML、Markdown、emoji 混淆、敏感词变体？
- 管理员能不能一键下架、封禁、追溯？

## 文件/图片

- 我能不能上传大文件拖垮存储/CDN？
- 我能不能上传非图片文件伪装成图片？
- 我能不能拿上传 URL 当公共图床？
- 我能不能让平台域名承载违规图片？
- 存储桶是不是 public-read？URL 有没有过期？

## AI / Agent

- 我能不能诱导 AI 说出系统提示、隐藏规则、内部接口或密钥？
- 我能不能让 AI 执行它不该执行的工具？
- 我能不能通过 RAG 读到其他用户的数据？
- AI 输出能不能直接发布违规内容？
- 有没有 token 成本上限和异常报警？

## 权限与数据

- 我能不能改 URL 里的 ID 读取/修改别人的资源？
- 写接口是否只依赖前端隐藏按钮？
- 普通用户能不能访问 admin API？
- CORS、Cookie、CSRF 设置是否允许跨站操作？
- 日志、错误、响应里有没有泄露 token、验证码、手机号、邮箱、内部路径？

## 运维与事故

- 出现垃圾内容、违法图片、短信轰炸、AI 失控时，谁会收到报警？
- 管理员能不能快速关闭相关功能？
- 能不能回滚？能不能追踪责任用户和请求来源？
- 有无灰度、白名单、feature flag？
