# 开工前项目准备清单

正式让 Codex 写代码前，必须检查：

- 是否有清晰项目目录
- 是否有 docs/
- 是否有 app/ 或 src/
- 是否有 design/
- 是否有 mock/ 或 mocks/
- 是否有 scripts/
- 是否有 README.md
- 是否有 progress.md
- 是否有 docs/spec.md
- 是否有 docs/acceptance.md
- 是否有 Git 仓库
- 是否有第一次 commit
- 是否有 .gitignore
- 是否已忽略 .env / .env.local
- 是否知道第一版 MVP 做什么
- 是否知道第一版不做什么

最低目录建议：

docs/
app/ 或 src/
design/
mocks/
scripts/
README.md
progress.md
