# Idea → MVP → Launch → Scale 风险地图

## Idea 阶段

风险：

- 问题是假问题
- 目标用户不清楚
- 替代方案已经足够用
- 用户不愿付费
- 只有自己觉得有需求

## MVP 阶段

风险：

- 一上来做太大
- 没有先做低成本验证
- 把原型当产品
- 把 UI 当价值
- 没有明确通过标准

## Launch 阶段

风险：

- 点赞不等于需求
- 夸奖不等于付费
- 试用不等于留存
- 早期热度不等于 PMF
- 没有反馈闭环

## Scale 阶段

风险：

- 成本失控
- 平台依赖
- 获客变贵
- 同质化严重
- 没有数据飞轮
- 没有流程护城河
- 创始人注意力成为瓶颈
