# 插件验证测试用例

测试用例 1：

输入：
使用 decision-stress-test-gate 帮我压力测试：我想做一个餐饮商家 AI 评论回复 Agent。先不要鼓励我。

期望输出必须包含：

- 决策压力测试 V2 报告
- 一句话定义
- 冷酷结论
- 最致命的 3 个假设
- 半年失败复盘
- VC / 风险审查 10 问
- 7 天现实验证计划
- 继续 / 调整 / 暂停建议
