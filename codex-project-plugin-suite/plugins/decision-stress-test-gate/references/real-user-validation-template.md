# 现实验证模板

7 天验证计划：

第 1 天：定义目标对象和核心假设
第 2 天：准备 1 页说明、原型或报价
第 3-5 天：访谈 5-10 个真实目标对象
第 6 天：整理反馈
第 7 天：判断继续、调整或暂停

访谈问题：

1. 你现在有没有这个问题？
2. 你多久遇到一次？
3. 你现在怎么解决？
4. 这个问题让你损失了什么？
5. 如果有这个方案，你会不会试？
6. 你愿意为它付多少钱、花多少时间或承担多少成本？
7. 你最担心什么？
8. 你是否愿意留下联系方式继续试用？

通过标准：

- 至少 5 个真实目标对象参与
- 至少 3 个承认痛点存在
- 至少 2 个愿意试用或继续沟通
- 至少 1 个愿意付费、合作、预约或采取下一步行动
