# 压力测试提示词库

## 致命假设法

帮我找出这个项目最致命的 3 个假设。
要求：只要其中一个假设错了，整个项目就会失败。
请说明每个假设为什么致命，以及我应该如何最快验证它。

## 半年失败复盘法

假设这个项目已经做了 6 个月，最后失败了。
请你写一份事后复盘，反推它失败的所有可能原因。
不要鼓励我，不要安慰我，只讲真实死因。

## VC 拷问法

你现在是一个极其挑剔的投资人。
请只看商业闭环，用最尖锐的方式质疑我的项目。
请提出 10 个最难回答的问题，并指出我现在最薄弱的地方。

## 禁止鼓励法

请不要鼓励我。
请不要说这个项目有潜力。
请只告诉我这个项目最可能死在哪一步。
一句话说透，不要给情绪价值。

## 现实验证法

请帮我设计一个 7 天验证计划。
目标是在不开发完整产品的情况下，判断是否有真实用户愿意试用、付费或继续沟通。
