# VC / 风险审查问题库

至少包含以下问题：

1. 目标用户到底是谁？
2. 他们为什么现在必须用这个方案？
3. 他们不用会损失什么？
4. 你怎么找到第一批用户？
5. 他们为什么愿意付费？
6. 现在他们用什么替代方案？
7. 你的方案和替代方案有什么本质区别？
8. 这个需求是否高频？
9. 这个需求是否刚需？
10. 如果平台规则变了怎么办？
11. 成本结构是什么？
12. 你能坚持多久？
13. 有没有合规风险？
14. 有没有隐私风险？
15. 你最怕哪一个假设被证伪？
16. 如果 7 天内找不到真实用户反馈，你还要不要继续？
17. 如果竞品免费，你怎么办？
18. 如果用户说“挺好但不付费”，你怎么办？

每个问题都要追问：为什么重要？当前最薄弱的地方是什么？
