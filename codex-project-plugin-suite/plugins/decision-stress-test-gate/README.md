# 决策压力测试器 V2

版本：2.0.0

作用：

在做产品、AI Agent、副业、账号、创业、投资、合伙、换工作、商业合作或重大决策前，先做冷静压力测试。

它会输出：

- 一句话定义
- 决策类型与风险等级
- 冷酷结论
- 最致命的 3 个假设
- 半年失败复盘
- VC / 风险审查 10 问
- 替代方案分析
- Idea → MVP → Launch → Scale 风险地图
- 7 天现实验证计划
- 风险评分表
- 继续 / 调整 / 暂停建议

适合场景：

- AI Agent 项目
- APP / 小程序
- SaaS
- 餐饮评论回复工具
- 周易玄学 APP
- 短视频自动化账号
- 自媒体账号定位
- 副业选择
- 投资决策
- 合伙决策
- 换工作
- 商业合作
- 任何重大决策

使用方式：

在 Codex 中输入：

使用 decision-stress-test-gate 帮我压力测试这个想法。

或者：

先不要鼓励我，帮我找出这个项目最可能怎么死。

试用提示词：

使用 decision-stress-test-gate 帮我压力测试：我想做一个餐饮商家 AI 评论回复 Agent。先不要鼓励我，重点找出最致命的 3 个假设、半年后失败原因、VC 拷问 10 问、Idea → MVP → Launch → Scale 风险地图和 7 天现实验证计划。
