# 竞品分析器

内部 ID：competitor-analyzer

## 用途

分析竞品、替代方案和差异化。

帮助项目判断竞品是谁、用户现在怎么解决、为什么要换、怎么定价、差异化是否足够强。

## 什么时候使用

当你需要处理与“竞品分析器”相关的项目工作时，在 Codex 对话里直接说：

使用 competitor-analyzer 帮我……

## 小白使用方式

1. 在 Codex 的 /plugins 里找到并启用这个插件。
2. 新开一个会话。
3. 复制默认提示词之一，让 Codex 按这个插件的规则输出。
4. 如果输出不够具体，继续要求“给我可复制的下一步命令”。

## 默认提示词

- 使用 competitor-analyzer 帮我分析竞品。
- 帮我分析这个产品的替代方案和差异化。
- 用户为什么会从竞品换到我这里？
