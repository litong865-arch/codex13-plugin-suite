# 用户访谈生成器

内部 ID：user-interview-generator

## 用途

帮你找真人验证需求。

生成 7 天用户验证计划、访谈问题、私信话术、问卷、反馈记录模板和通过标准。

## 什么时候使用

当你需要处理与“用户访谈生成器”相关的项目工作时，在 Codex 对话里直接说：

使用 user-interview-generator 帮我……

## 小白使用方式

1. 在 Codex 的 /plugins 里找到并启用这个插件。
2. 新开一个会话。
3. 复制默认提示词之一，让 Codex 按这个插件的规则输出。
4. 如果输出不够具体，继续要求“给我可复制的下一步命令”。
