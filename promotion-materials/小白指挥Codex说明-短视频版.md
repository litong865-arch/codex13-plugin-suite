# 小白指挥 Codex 说明-短视频版

这套提示词是给技术小白准备的：你不需要先懂完整开发流程，先让总控判断下一步。

## 开场提示词

```text
先不要写代码，请使用 项目总控闸门 帮我总控这个项目。

如果需要 内置浏览器、谷歌浏览器、电脑操作 或子线程，请自动给出调用计划。
如果当前工具不可用，请明确告诉我缺什么。
```

## 项目描述模板

```text
我想做一个【项目类型】。
目标用户是【谁】。
核心功能是【功能】。
未来可能上线/交付/收费。

请判断：
1. 当前项目处于什么阶段
2. 应该先调用哪些插件
3. 每一步复制什么提示词
4. 哪些地方会阻塞
5. 下一步最小行动是什么
```

## 如果 Codex 直接写代码

```text
停一下。先不要写代码。
请回到 项目总控闸门 的总控流程，先输出项目总控闸门报告。
```
