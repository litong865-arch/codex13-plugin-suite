# 视觉风格说明

## 画布

- 尺寸：1080x1920
- 比例：9:16
- 用途：抖音图文卡片

## 风格

- 深色科技产品感
- 知识博主图文感
- 目标人群：技术小白、想用 Codex 做项目但不熟悉开发流程的人
- 大标题 + 观点句 + 插件标签 + 底部记忆点
- 当前导出版使用纯色深色背景，避免复杂背景干扰阅读
- IMGA2/图像生成底图仅作为备用素材保留
- 全套统一使用青色主色，不再按页面切换红/橙/绿/紫
- 所有普通页统一使用相同卡片、标签和底部记忆框样式

## 配色

- 背景：#060E1C
- 面板：#0D1B30
- 青蓝：#22D3EE
- 紫色：#A78BFA
- 绿色：#34D399
- 橙色：#F59E0B
- 红色：#F87171

## 字体

- Microsoft YaHei
- 标题粗体，正文常规

## 禁止

- 不复制参考图文原文
- 不使用对方头像或排版
- 不夸大说替代 虚拟团队 / 能力约束
- 不把一页塞满所有插件
- 不把插件 slug 直接展示给小白用户，统一使用中文展示名
